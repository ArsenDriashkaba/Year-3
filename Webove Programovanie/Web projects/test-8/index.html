1. Start drawing by clicking on the board.
    The last point is always highlighted with a green square.
  2. To save your drawing type the name and hit "Save" button.
    All saves you can find at "saves" folder at the project directory.
  3. To load you saved drawing hit "Choose File" button,
    and find the folder described in the previous point
    Then just hit the "Apply" button and go on :) Enjoy!
  4. To clear the board just hit the "Clear board" button.
